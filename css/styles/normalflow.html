normal flow doc - Python Packaging User Guide

Installing from PyPI

python3 -m pip install "SomeProject"

Requirements files

python3 -m pip install -r requirements.txt

Installing from VCS

python3 -m pip install -e SomeProject @ git+https://git.repo/some_pkg.git

Installing from Indexes

python3 -m pip install -e SomeProject @ git+https://git.repo/some_pkg.git python3 -m pip install --index-url http://my.package.repo/simple/ SomeProject

Installing from a local src tree

python3 -m pip install -e python3 -m pip install
